The Basic Language Of the Web: HTML

posted by Laura Jones on Monday, June 21st 2027

All modern websites and web applications are built using three fundamental technologies: HTML, CSS and JavaScript.These are the languages of the web.

In this post let's focus on HTML.we will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for Hyper Text Markup Language.It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language).

HTML consists of elements that describe different types of content:paragaraphs,links,headings,images,video,etc.Web browsers understand HTML and render HTML code as websites.

in HTML, each element is made up of 3 parts.

  1. The opening tags
  2. the closing tags
  3. the actual elements

you can learn more about web at MDN Web Docs

Why should you learn HTML

their are countless reason for learning fundamental Languages of the web. Here are 5 of them:

  • To be able to use fundamental web dev language
  • To hand-craft beautiful websites instead of relying on tools like Wordpress or Wix
  • To build web applications
  • To impress friends
  • To have fun 😃
